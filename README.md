## Setup à¤•à¥ƒà¤·à¤¿Care Locally
Once you have followed the steps mentioned in [Contribution](https://github.com/pukhrajkushwah/SIH24_nextgencoders/blob/master/Contributing.md) to fork and clone the repository, do the following to setup à¤•à¥ƒà¤·à¤¿Care locally on your PC.
```bash
cd web
```
Install all the dependencies
```bash
npm ci
```
Add the API key to the `.env` folder as mentioned in `.env.example`
Get the Plant Id API key from [here](https://web.plant.id/plant-identification-api/)
```
NEXT_PUBLIC_PLANT_ID_API_KEY = "Your API Key"
```

Run the server locally
```bash
npm run dev
```
That's it you are good to go! Looking forward to your contributions.
